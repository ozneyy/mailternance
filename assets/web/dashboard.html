À propos du fichier .env :

Ces paramètres sont stockés localement dans le fichier caché .env situé à la racine du projet.

En enregistrant les modifications, le serveur met à jour le fichier et applique immédiatement les nouveaux accès SMTP et IMAP sans nécessiter de redémarrage de l'application (sauf pour le port d'écoute HTTP).

Les variables de fusion vous permettent de personnaliser automatiquement le contenu de vos e-mails pour chaque destinataire de votre fichier CSV.

Variables de contact
  • {[ varsHelp.FirstName ]} : Insère le prénom du destinataire.
  • {[ varsHelp.LastName ]} : Insère le nom du destinataire.
  • {[ varsHelp.Company ]} : Insère le nom de l'entreprise.
  • {[ varsHelp.Position ]} : Insère le poste du contact (ex: RH).
  • {[ varsHelp.SenderName ]} : Insère votre nom d'expéditeur configuré dans les paramètres.
Variables de liens configurés

Dans l'onglet de droite, vous pouvez créer des liens nommés (ex: Portfolio, Github). Pour les insérer :

  • Utilisez la syntaxe {[ varsHelp.LinksVotreClef ]} (ex: {[ varsHelp.LinksPortfolio ]}).
  • Dans un mail HTML : Cela générera automatiquement un lien hypertexte bleu et cliquable avec le texte du lien.
  • Dans un mail Texte brut : Cela affichera le texte du lien suivi de l'URL entre parenthèses.
